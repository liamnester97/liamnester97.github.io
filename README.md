# liamnester97.github.io

Personal site for Liam Nester, live at https://liamnester97.github.io/.

Plain static HTML and CSS with no build step or JavaScript. GitHub Pages serves the files in this repo directly (see `.nojekyll`, which disables Jekyll processing).

Pages:
- `index.html`, about/home
- `experience.html`, career timeline
- `projects.html`, project portfolio (in progress)
- `resume.html`, resume
- `contact.html`, contact links

To preview locally, open `index.html` directly in a browser or serve the folder with any static file server.
